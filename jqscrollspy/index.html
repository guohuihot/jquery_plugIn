参数 默认值 版本 说明 详细
obj li 1.0 (必须)切换对象:
效果中要切换的对象
使用: {obj:'li'} 		 
cell null 1.0 控制切换效果的外层: 插件会自动去找该内部的子元素作为控制切换的对象;如果子元素为空, 插件会自动往内部增加对应的子元素, 例如:下面的div子元素为空 插件执行后, 会生成 使用: {cell:'.dot'}  
trigger mouseover 1.0 (必须)切换效果的触发方式:
参数: {effect: 'mouseover' | 'click'}
说明：{effect: 鼠标悬停到cell对象上时触发动画切换到对应的obj对象上 | 鼠标点击到cell对象上时触发动画切换到对应的obj对象上}
使用: {trigger : 'mouseover'} 		 
effect fade 1.0 (必须)切换效果:
参数: {effect: fade | fold | top | topLoop | topMarqueue | left | leftLoop | leftMarqueue | weibo}
说明：{effect: 渐显 | 淡入淡出 | 上滚动 | 上循环滚动 | 上无缝滚动 | 左滚动 | 左循环滚动 | 左无缝滚动 | 微博效果}
使用: {effect : 'fade'} 		 
speed 400 1.0 (必须)切换效果的速度:
说明: 毫秒(ms),一次效果用的时间
使用: {speed:400} 		 
index 0 1.0 (必须)索引值:
说明: 就是初始高亮的值,从0开始,例如我想让幻灯片从第二张开始切换可以这样: {index:1}
使用: {index:0} 		 
autoplay 1 1.0 (必须)自动切换:
说明: 是否自动切换到下一个, 1:自动切换, 0:不切换, 为0时"interval"设置无效
使用: {autoplay:1} 		 
interval 3000 1.0 自动切换:
说明: 毫秒(ms), 自动切换到下一个的间隔时间, effect: top(left)Marqueue时表示滚动的速度
使用: {interval:3000} 		 
prevbtn null 1.0 上一个按钮:
说明: 切换到上一个的按钮
使用: {prevbtn:'.prev'} 		 
nextbtn null 1.0 下一个按钮:
说明: 切换到下一个的按钮
使用: {nextbtn:'.next'} 		 
delay 150 1.0 触发效果后的延迟时间:
说明: 毫秒(ms), 点击(鼠标悬停到)cell的对象后将要触发效果前的时间, 避免重复执行以及性能上考虑设置150ms
使用: {delay:150} 		 
easing swing 1.0 动画的缓动效果:
说明: 可选值 swing | linear, 若要实现更多效果需要jQuery动画效果扩展增强插件(jquery.easing.js)支持
使用: {easing:'swing'} 		 
visible 1 1.0 (必须)obj对象的可见个数:
说明: 插件会自动计算出外层的大小,从而显示对应的设置大小，effect: fade | fold时无效
使用: {visible:1} 		 
steps 1 1.0 (必须)每次滚动的数量:
说明: 每次滚动的数量,设置时建议 steps <= visible, effect: fade | fold | top(left)Marqueue时无效
使用: {steps:1} 		 
overstop 1 1.0 (必须)鼠标悬停是否停止动画:
说明: 到obj,cell,prevbtn,nextbtn时, 是否停止动画, top(left)Marqueue时 prevbtn,nextbtn无效
使用: {overstop:1} 		 
showtit 0 1.0 是否显示标题: 说明: 插件会自动创建个标签a,其内容和链接分别调用当前激活的obj对象的data-title和data-url, 例如:obj对象为 会对应的生成(该设置只有当visible: 1时有效) 使用: {showtit:1}  
pagewrap null 1.0 分页页码对象: 说明: 与cell一样, 插件会自动给pagewrap标签内添加对应的页码, 例如: 使用: {pagewrap:'.pagewrap'}  
actclass act 1.0 (必须)cell当前激活对象的class:
说明: 默认为'act', 有时我们布局时可能用的class不是act,而又懒得去改,插件就可以直接设置, 例如: actclass: 'current'
使用: {actclass:'.act'} 		 
btnLoop 1 1.0 按钮循环:
说明: 点击前/后一个时,到第一(最后)时,按钮会添加'disabled'样式,并且按钮不可用； 使用: {btnLoop:1} 		 
wheel 0 1.0 鼠标悬停对象上时滚动鼠标是否切换:
说明: 需要插件jquery.mousewheel.min.js支持；版本3.1.0以上
使用: {wheel:1}
